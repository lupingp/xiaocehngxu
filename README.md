### 本地存储调用说明：
##### `主要起到提高体验度、减轻服务器压力`
| 数据 | 调用页 | 函数名 | 存储条件 | 有效时间 |
| :--------| :-------- | :-------- | :--------| :------: |
| 用户数据[账号/token/主账号] | account | settingAccount | 无数据/数据过期 | 3天 |
| 部门信息[主部门/子部门/科室] | app | onLaunch | 无数据/数据过期 | 1小时 |
| 专家列表[专家信息] | search | onLoad | 专家咨询 | 1小时 |
